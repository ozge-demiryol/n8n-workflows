## 🛠️ How to Use

1.  Navigate to the folder of the workflow you are interested in.
2.  Open the folder and download the `.json` workflow file.
3.  In your n8n instance, click on **"Import from File"** (or simply copy-paste the JSON content into the n8n canvas).
4.  Configure the **Credentials** for the specific nodes as described in the sub-folder's README.
